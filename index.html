Git

GIT - система управления версиями, или контроля версиями, записывает историю изменения файла или набора файлов, чтобы в будущем была возможность вернуться к конкретной версии

Комманды:

  • $ git init
  • $ git add .
  • $ git status
  • $ git commit -m 'комментарий'
  • $ git push -u hub master
